Игра SNAKE на python, написанная Озорниным Ярославом.
Игра запускается ТОЛЬКО через терминал через команду python scr/Main.py, также
можно игру запустить с аргументом -h чтобы посмотреть краткую инструкцию.

Для управления змейкой надо использовать кнопки:
    1. KEY_UP      - Повернуть наверх
    2. KEY_DOWN    - Повернуть вниз
    3. KEY_LEFT    - Повернуть влево
    4. KEY_RIGHT   - Повернуть вниз
    5. q/й         - Выйти из игры
